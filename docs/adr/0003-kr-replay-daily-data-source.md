# ADR 0003 — KR 리플레이 과거 일봉 데이터 소스 = 공공데이터포털 금융위 주식시세정보(15094808)

- 상태: 채택 (2026-07-07)
- 맥락: 리플레이(과거장 훈련소)는 과거 일봉을 정적 JSON으로 **public repo에 커밋**해 서빙한다(`web/public/replay/**`). 따라서 데이터 소스는 **재배포(공개 게시)를 허용하는 라이선스**여야 한다. KR 과거 일봉을 어디서 받아 커밋할지가 이 결정의 대상. (ADR 0002 표의 "리플레이 데이터 KR=공공데이터포털" 항목을 실제 데이터셋·근거 수준으로 확정한다.)

## 결정

리플레이용 KR 과거 일봉은 **공공데이터포털 금융위원회_주식시세정보 API(데이터셋 15094808)**로 수집해 정적 JSON을 repo에 커밋한다. 이미 워커에 있는 **KIS 앱키 재사용은 기각**한다.

- 수집: `getStockPriceInfo` + `beginBasDt`/`endBasDt`로 과거 기간별 일별 시세 조회(영업일+1 갱신 = 실시간 아님, 리플레이용으로 문제없음).
- 인증키: `DATA_GO_KR_SERVICE_KEY` env(수집 스크립트 1회성용). 하드코딩 금지, 값은 env로만.
- 출처: data.go.kr/data/15094808/openapi.do

## 대안과 기각

- **(기각) KIS Open API 재사용** — 워커에 이미 KIS 앱키가 있어 재사용이 편하고, **기술적으로도 가능**하다: KIS는 기간별 일봉을 REST로 제공한다(`inquire-daily-itemchartprice`, TR `FHKST03010100`, `FID_INPUT_DATE_1/2`로 기간 지정, 호출당 100건, 토큰 `/oauth2/tokenP`). 그러나 KIS Open API는 개발자에게 시세 **재배포 권리를 부여하지 않는다** — 시세정보는 "투자 목적·본인 한정, 제3자 제공 불가"가 원칙이고 원천 데이터 IP는 한국거래소(KRX)다. 일봉 JSON을 public repo에 커밋하면 제3자 제공/공개 게시에 해당해 약관 위반. **즉 능력이 아니라 라이선스 문제** — "받을 수는 있으나 커밋할 권리가 없어" 부적합.
  - 출처: KIS 개발자센터 apiportal.koreainvestment.com/about-open-api, KRX 마켓데이터 이용약관 data.krx.co.kr.
  - **정직한 한계(명시)**: ToS 전문이 JS 렌더링/403이라 정확한 "제○조" 조항 번호까지는 확인하지 못했다. 정책 방향(제3자 제공 불가·KRX IP·재분배 금지)만 공식 출처에서 일관 확인됨.
- **(기각) yfinance · Alpha Vantage** — 약관상 재배포 금지.
- **(기각) Finnhub 캔들** — 과거 캔들은 유료 전환.
- **(기각) Stooq KR** — 재배포 라이선스가 KIS처럼 애매 + KR 커버리지 미확인. (US는 Stooq를 "사실 데이터 소량 + 출처 표기"로 이미 채택 — PRD §8. KR은 공공데이터포털의 명시적 '제한 없음'이 더 깨끗해 이쪽으로.)

## 근거

- 데이터셋 15094808 상세페이지 **이용허락범위 = "제한 없음"**(재배포·공개 게시 명시 허용) → repo 커밋 합법 확정. 이것이 KIS 대비 결정적 차이다.
- 영업일+1 갱신(실시간 아님)이지만 리플레이는 과거 일봉만 쓰므로 신선도 요구가 없어 무관.
- **주의: 반드시 15094808을 쓸 것.** 같은 계열 '주식발행정보'는 이용허락범위가 **제2유형(상업 이용금지)**이라 재배포 불가.

## 영향

- KIS는 실시세 WS(H0STCNT0) 전용으로 계속 사용(워커, ADR 0001·0002). KR 과거 일봉만 공공데이터포털로 이원화.
- 수집 키 `DATA_GO_KR_SERVICE_KEY`는 워커 env(`worker/.env.example`)에 자리만 추가(1회성 수집 스크립트용, 런타임 상시 의존 아님).
- 미수집 심볼은 `web/public/replay/covid-2020/manifest.json`의 `symbols.KR`가 pending으로 이 ADR을 가리킨다.
